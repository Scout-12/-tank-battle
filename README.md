# 坦克大战 🎮 Tank Battle

经典坦克大战游戏的 HTML/JavaScript 实现，受 NES 游戏 **Battle City**（坦克大战）启发。

## 玩法

- **目标**：保卫你的基地（右下角旗帜），摧毁所有敌方坦克
- **生命**：3 条命，被击中后短暂复活
- **得分**：每击毁一辆敌军坦克得 100×关卡数 分

## 操作

| 按键 | 操作 |
|------|------|
| `↑↓←→` 或 `WASD` | 移动坦克 |
| `空格` | 射击 |

## 特性

- 砖墙（棕色）可被子弹摧毁
- 钢墙（灰色）不可摧毁
- 敌军 AI 会自动追踪玩家并射击
- 每关 20 辆敌军坦克，分波次出现
- 粒子爆炸特效

## 运行

直接在浏览器中打开 `index.html` 即可开始游戏。

## 技术栈

纯 HTML + Canvas + JavaScript，无任何外部依赖。
